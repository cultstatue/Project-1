  # Ready?
 [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ## Description
ready? Is an application made to simplify the planning process of going out by providing a list of ticket master events as well as a local forecast for a given US zip code.
